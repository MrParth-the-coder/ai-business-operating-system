# 📖 About AI-BOS

**AI-BOS (AI Business Operating System)** is an AI-powered web-based Business Management Platform developed as a Final Year Engineering Project.

The system helps small businesses automate daily operations through modern web technologies, Artificial Intelligence, Machine Learning, and business analytics.

AI-BOS provides a centralized platform where business owners can manage:

- 📦 Inventory
- 💰 Sales
- 👥 Customers
- 🚚 Suppliers
- 👨‍💼 Employees
- 📊 Reports
- 🤖 AI Business Assistant
- 📈 Machine Learning Predictions

---

# 🎯 Project Objectives

- Automate business operations
- Reduce manual work
- Improve business decision making
- Predict future sales
- Analyze customer behavior
- Provide AI-powered business insights
- Generate reports automatically

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/MrParth-the-coder/ai-business-operating-system.git
```

---

## Backend Setup

```bash
cd backend

python -m venv .venv

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
